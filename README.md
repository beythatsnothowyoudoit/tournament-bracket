# 16강 토너먼트 대진표 프로그램

이 프로젝트는 16개 팀이 참가하는 토너먼트 대진표를 생성하고, 경기 결과를 입력하며, 자동으로 다음 라운드로 진출시키는 Python 기반 프로그램입니다.

## 기능
- 16개 팀 등록
- 자동 대진표 생성
- 경기 결과 입력
- 다음 라운드 자동 진출
- 최종 우승자 표시

## 실행 방법
```bash
git clone https://github.com/사용자명/tournament-bracket.git
cd tournament-bracket
pip install -r requirements.txt
python main.py
예시
프로그램 실행 후, 각 경기의 승자를 입력하면 자동으로 다음 라운드로 진출합니다.
